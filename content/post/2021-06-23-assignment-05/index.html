firstly, we want to know the distribution for all diamonds.

diamonds %>% 
  ggplot(aes(x = price, group = cut, fill = cut)) +
  geom_histogram(binwidth = 10)

Then, we might want to see how the quality of the cut, or the color, or the clarity, affects the price.

ggplot(diamonds, aes(x=carat, y=price, color=clarity)) + geom_point()

Thirdly, we could also improve the data representation by altering the transparency of the data points. What this does is showing us where the majority of the data points sit:The more data points that sit in a particular location (the darker the area), the more observations there are.

  • Note:lower alpha values increase transparency of points
diamonds %>% 
  ggplot(aes(x = depth, y = price)) +
  geom_point(alpha = 0.1)

From the above plot, it is easy to find that most diamonds have the depth ranging from 55-65mm.

## Create Correlation Matrix among variables.

diamonds %>%
    mutate_if(is.factor,as.numeric) %>%
   select(price,everything()) %>%
    ggcorr(method = c("everything", "pearson"),label = T)

From the matrix above, we found there are some highly-related variables.

Thus, we could predict that there may be non-linear relationship between carat with price from the chart below.

diamonds %>%
ggplot(aes(x=carat,y=price))+
  scale_y_continuous(trans=log_trans(),labels=function(x) round(x,-2))+
  geom_point()+
  geom_smooth(method="lm",formula="y~poly(x,4)")+
  theme_bw()

However, if we would rather show a best fit straight line rather than a curve, we can change the option in the geom_smooth layer.

ggplot(diamonds, aes(x=carat, y=price)) + geom_point() + geom_smooth(se=FALSE, method="lm")
## `geom_smooth()` using formula 'y ~ x'
